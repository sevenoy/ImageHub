# ImageHub 图片综合工具

ImageHub 是一个本地图片综合工具，当前包含多图合成与批量水印功能。

原项目显示名为 InstaGrid；当前仅更改展示名称，项目目录仍保留为 `InstaGrid-main`。

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
